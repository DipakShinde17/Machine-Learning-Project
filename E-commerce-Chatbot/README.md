
💡 Project Description

The AI Chatbot for Customer Support is a web-based application designed to automate responses to customer queries using Natural Language Processing (NLP) techniques.

The system uses a pre-trained machine learning model (resume.pkl) to understand user inputs and provide relevant responses. It is built using a Python-based backend and a simple frontend interface.

⚙️ Key Features

💬 Real-time chatbot interaction

🧠 NLP-based query understanding

⚡ Instant automated responses

🌐 Web interface using HTML (templates folder)

🎨 Static assets (CSS/JS) for UI design

🤖 Machine Learning model integration (.pkl file)


🛠️ Technologies Used

Python

Flask (Backend framework)

HTML, CSS (Frontend)

Machine Learning / NLP

Pickle (.pkl model file)


🔄 Working Flow

User enters a query in the chatbot UI

Request is sent to the backend (app.py)

The ML model (resume.pkl) processes the input

Chatbot predicts the intent and generates a response

Response is displayed to the user instantly


🎯 Outcome

Reduced manual customer support effort

Faster response time

Improved user experience

Scalable solution for handling multiple users
